Général

Détails

  • Afficher les erreurs : met en évidence les ruptures dans la hiérarchie de titres.
  • Mettre en évidence : entoure (pendant quelques secondes) le titre dans la page lorsqu’un élément de l’arborescence est sélectionné.
  • Afficher les attributs ARIA : si des attributs aria-label ou aria-labelledby sont présents sur les titres, l’outil affiche l’attribut et la valeur (car les lecteurs d’écran y accèdent et les utilisent).
  • Afficher les ancres : affiche une ancre sur les titres qui le sont.
  • Thème sombre : lorsque l’option est cochée, les styles de l’extension sont modifiés. Le fond est sombre et la couleur de texte claire.
  • Enregistrer la largeur lors du redimensionnement : enregistre la nouvelle largeur après un redimensionnement, de sorte que la prochaine fois que l’extension est utilisée, il conserve cette largeur.
  • Renvoyer à la ligne : par défaut, le texte des éléments de l'arborescence est tronqué. Lorsque cette option est activée, le texte est affiché sur plusieurs lignes si nécessaire.
  • Afficher une infobulle d’informations : affiche une infobulle avec des informations au survol des éléments de l’arborescence.
  • Lier à la fenêtre d'affichage : affiche un fond clair sur les titres visibles dans la zone affichée par le navigateur.
  • Langue : changer la langue. Le changement sera effectif après réouverture de l’extension.

Détails

  • Afficher une erreur si pas de H1 au début : affiche une erreur si le premier titre n’est pas un H1. L’option « Afficher les erreurs » doit être cochée.
  • Afficher le niveau de titre : affiche le niveau de titre avant le texte.
  • Afficher les titres vides : affiche les titres vides dans la hiérarchie de titres.
  • Afficher les titres masqués : affiche tous les titres même ceux qui sont masqués.
Arborescence de l'outline HTML5

Il n'existe aucun navigateur ou technologie d'assistance doté d'implémentations natives de l'algorithme de l'arborescence W3C. Par conséquent, on ne peut pas l’utiliser pour transmettre la structure du document aux utilisateurs, et les auteurs devraient utiliser le niveau des titres pour le faire (HTML 5.2 Création d'une arborescence).

*Lorsque l'option « Afficher les sections masquées » est activée, l'analyse peut avoir des impacts sur les performances dans les documents longs.

Détails

  • Activer l’analyse : l'extraction de l'arborescence du document peut avoir un impact sur les performances, donc son exécution est facultative et désactivée par défaut. Lorsqu'elle est activée, les deux analyses (titres et arborescence HTML 5) apparaissent comme des onglets dans le panneau.
  • Afficher l'index de section : affiche l'index de section avant son texte.
  • Afficher la balise section : inclut la balise de section en tant que partie du contenu dans l'arborescence.
  • Afficher les sections masquées : identifie et affiche les informations de toutes les sections, qu'elles soient masquées ou non.
  • Afficher HGROUP : avec cette option, l'analyse considère l'élément HGROUP comme un élément de section.
